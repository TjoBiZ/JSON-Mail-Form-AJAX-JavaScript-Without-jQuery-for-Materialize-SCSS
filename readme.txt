Форма обратной связ на фреймворке Materialize. Моя поддержка только на платной сонове.
 Скрипт собирает всю необходимую информацию о пользователе для удачной работы менеджера, который реагирует на присланную информацию из форм.
  Работает со всем типами форм на данном фреймоврке. 
  Сделано на чистом JavaScript без дополнительной библиотеки jQuery, которая нагрузит Ваш сайт.
  
  Есть вторая версия скрипта на более современном стандарте и использованием объекта FormData, которую можно отыскать в моих других рипозиториях.
